# 🗄️ Banco QUIZ

Projeto desenvolvido durante as aulas de Banco de Dados com o objetivo de praticar modelagem de dados, criação de tabelas e relacionamentos utilizando SQL.

O sistema foi estruturado com foco em um projeto de QUIZ educacional, aplicando conceitos fundamentais de banco de dados relacional e organização de informações.
